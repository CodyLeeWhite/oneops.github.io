OneOps

OneOps enables continuous lifecycle management of complex, business-critical application workloads on any cloud-based infrastructure.

OneOps Administrator

The OneOps Administrator deploys and configures One Ops software for the organization. Initial bootstrapping of one ops can be done following the getting started.

By configuring Clouds, Organizations and other components, the building blocks are put in place for the end Users to configure and deploy their applications.
